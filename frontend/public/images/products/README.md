# Product Images

This folder should contain product images for your store.

## Instructions

1. Delete this README.md
2. Add your product images (recommended size: 300x300px, WebP format)
3. Update product data in the backend database

## Recommended Format

- Format: WebP or PNG
- Size: 300x300px minimum
- Naming: Use product SKU or ID (e.g., `product-001.webp`)

## Example

```
images/products/
├── product-001.webp
├── product-002.webp
├── category-fruits/
│   ├── apple.webp
│   └── banana.webp
└── ...
```
